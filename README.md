# helm-charts
## 打包
1. `helm package postgresql-ha`
2. `helm repo index --url https://github.com/zhl003/helm-charts.git/ .`
## 部署
1. `helm repo add zhlrepo https://zhl003.github.io/helm-charts/`
2. `helm install zhlrepo/postgresql-ha`
